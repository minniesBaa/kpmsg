KP's Message Board is an ephemeral message board app built with Python Flask and hosted on Google Cloud Run. Instead of an account, 
we use each user's IP address (or, if IPv6, the first 15 characters of it) (which [isn't secret anyway](https://www.bigdatacloud.com/blog/understanding-ip-addresses)), a random 
number generated by your computer and saved in your browser, and a customizable signature that displays at the end of your posts.
